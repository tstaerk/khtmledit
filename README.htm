khtmledit is a wysiwyg html editor that shows you as well html plain text as html wysiwyg.

Example

You edit an html file. In the upper widget, you have it displayed as it will look like in a browser. In the lower widget, you see the html source code.

You can edit this page now. You can also access (and modify) html files directly on your server, provided the server is reachable via ssh.

Let's imagine you want to change index.htm on your webserver named yourserver.yourdomain.com. You can do it like this:

khexedit fish://youruser@yourserver.yourdomain.com/srv/www/htdocs/index.htm

You will be prompted for youruser's password and be able to change the file as if it was local.

Installing

To download, compile, link, install and run khtmledit, best do the following

- prepare the build

Set up a build environment as described at http://techbase.kde.org/Getting_Started/Build/KDE4. Become the user kde-devel:

ssh -X kde-devel

- cd into khtmledit's source code directory

- build khtmledit

cmake . && make && make install

- run it

khtmledit

Features

- wysiwyg

- fish:// support
